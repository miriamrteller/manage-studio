## Critical Constraints

1. **NO `any` types, NO `!` assertions** — Use `unknown` or explicit types
2. **RTL STRICT** — Only logical CSS (ms-, me-, ps-, pe-). NO physical (ml-, mr-, left, right)
3. **Accessibility MANDATORY** — WCAG 2.1 Level AA (Israeli law requirement)
4. **RLS non-negotiable** — Every Phase 1B/1C query must use auth.uid() + tenant_id
5. **Zod everywhere** — All external inputs validated
6. **PII Protection** — No raw student data in logs/code
